# Ant Colony Traveling Salesman Problem

Travelling salesman problem solved with Ant Colony Optimization using Go.

## How to run
- install [dep](https://github.com/golang/dep)
- `dep ensure`
- `go build tsp.go`
- `tsp.exe run -file=Filename.tsp`
